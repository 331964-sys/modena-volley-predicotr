# Modena Volley – Attack Predictor

Web app in Flask che predice la probabilità che un attacco di Modena Volley si trasformi in punto, 
sulla base di alcune caratteristiche tecniche dell'azione.

## Tecnologie usate

- Python 3
- Flask
- scikit-learn
- pandas, numpy
- HTML + CSS

## Come eseguirla in locale

1. Crea e attiva un ambiente virtuale (opzionale ma consigliato):

   ```bash
   python -m venv .venv
   .venv\Scripts\activate
2. Installa le dipendenze:
    pip install -r requirements.txt
3. Avvia l'app:
    python app.py
4. Apri il browser su:
    http://127.0.0.1:5000
